# CPP_00

CPP_00 est un projet du cursus 42.
Le but est de se familiariser avec les **bases du C++**, la création de classes et la manipulation d’objets.

## Compilation

```bash
make
```

## Contenu

* Création de **classes simples** avec constructeurs et destructeurs
* Gestion des **attributs et méthodes**
* Compréhension des concepts de **POO (programmation orientée objet)**
